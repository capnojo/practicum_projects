# Изучение рынка заведений общественного питания Москвы

# Описание проекта
Инвесторы фонда Shut Up and Take My Money планируют открыть заведение общественного питания в Москве и просят провести исследование рынка для выбора оптимального формата, локации и ценовой политики. Проект анализирует данные о кафе, ресторанах и барах Москвы, собранные из Яндекс Карт и Яндекс Бизнес летом 2022 года. Полученные результаты помогут инвесторам принять обоснованное решение о запуске нового заведения.
