# Поиска инсайтов и точек роста «Всё.из.кафе»

# Описание проекта
В сервисе доставки еды «Всё.из.кафе» найти инсайты для повышения удержания пользователей и улучшения взаимодействия с партнёрами-ресторанами. Проект изучает поведение пользователей, их путь в сервисе и точки, где они «теряются». Полученные выводы помогут оптимизировать работу с клиентами и партнёрами для роста показателей сервиса.
