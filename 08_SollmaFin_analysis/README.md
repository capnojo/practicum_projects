# Исследование результатов А/В-теста и поиск инсайтов для SollmaFin

# Описание проекта
В мобильном приложении SollmaFin проводится A/B-эксперимент нового онбординга, который обучает пользователей различиям между финансовыми активами и связанным с ними риском. Цель — оценить влияние обновлённого онбординга на депозитную активность пользователей, включая конверсию в первый и второй депозит и средние суммы депозитов. Результаты эксперимента помогут оптимизировать пользовательский опыт и повысить вовлечённость и монетизацию приложения.
