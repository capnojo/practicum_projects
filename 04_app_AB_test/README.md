# Разработка A/B-тестирования и анализ результатов

# Описание проекта
В компании-разработчике развлекательного приложения с «бесконечной» лентой проводится A/B-тест нового алгоритма рекомендаций, который должен показывать более интересный контент пользователям. Приложение использует две модели монетизации: подписку без рекламы и показ рекламы для остальных пользователей. Задача аналитика — рассчитать параметры эксперимента и проанализировать его результаты для проверки гипотезы команды разработчиков.
