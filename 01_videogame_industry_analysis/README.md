# Анализ развитие игровой индустрии с 2000 по 2013 год для команды игры «Секреты Темнолесья»

## Описание проекта
Команда игры «Секреты Темнолесья» готовит статью о развитии игровой индустрии в начале XXI века для известного IT-ресурса, чтобы привлечь внимание любителей старых игр и заинтересовать их своей игрой. В статье будет проведён обзор игровых платформ, жанров и региональных предпочтений игроков с акцентом на RPG. Для этого проект изучает исторические данные о продажах игр, пользовательских и экспертных оценках за период с 2000 по 2013 год.
