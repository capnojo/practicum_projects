# Анализ системы метрик приложения Procrastinate Pro+

# Описание проекта
В приложении Procrastinate Pro+ запущен новый спортивный контент для привлечения аудитории, увлекающейся спортом и здоровым образом жизни, с обновлённой маркетинговой стратегией. Пользователи выбирают различные пакеты контента, которые различаются по цене и объёму. Задача аналитика — изучить причины стагнации выручки после запуска нового контента и роста интереса к продукту.
