# Проверка гипотезы в Python и составление аналитической записки

# Описание проекта
Проверка гипотезы о том, что пользователи из Санкт-Петербурга проводят в среднем больше времени за чтением и прослушиванием книг в приложении Яндекс Книги, чем пользователи из Москвы. Для этого была использована односторонняя проверка статистической гипотезы с двумя выборками.
